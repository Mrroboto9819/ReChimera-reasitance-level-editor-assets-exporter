# Public docs (in development)

This folder is reserved for the **non-technical, end-user-facing
documentation** of ReChimera — written for people who just want to use
the app, not understand the parsers underneath.

The plan:

- "Getting started" — install, open a level folder, what each panel does.
- "Common workflows" — preview a moby, export a GLB to Blender,
  rip a sound to WAV.
- "FAQ" — game compatibility, missing files, why some textures look low-res, etc.
- Localised versions (English first, then the same languages the app UI ships in).

If you're a developer or you want to know *how* something works, see
[`../internal/`](../internal/) instead — that's where the parser
internals, the IGHW format reference, the GLB-export pipeline, the
cache schema, and the IT cross-references live.

> 🚧 **Status: in development** — currently empty. Once the in-app
> documentation viewer (Help → Documentation) ships, this folder will
> hold the same Markdown files the viewer reads, so they're available
> both offline-in-app and online on the project's site.
